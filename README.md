# R-File-manager-CLI-BASED-
R file manager a very heavy WIP and currently CLI based file manager written in Python made for linux
In order to run it (it hasnt been fully completed yet) open a terminal enter the folder where you downloaded it and run : python file_ops.py
Navigation :
uses commands like :

cd - moving around folders


cd.. - moving up a folder


pwd - shows your current location (path)


ls - lists the files in the folder/directory you are in


copy - copies a file  for example : copy test.txt


paste - pastes a file for example: paste test.txt


cut- cuts a file for example : cut test.txt


delete - deletes a file example: delete test.txt


zip- Archives/zips a file example : zip Test


mkdir - creates a folder


touch - creates an empty file


show_banner- prints the welcome banner


cd or cd ~ takes you to home


empty_trash - empties the trash folder


open - opens a chosen file example : open test.txt


exit - exits the script
